xmas lights

March 2020 - Present
Internet Controlled Lights!
GitHub: crhaugen/xmas-lights


Using a Raspberry Pi and Python to control a LED NeoPixel strip.
It was difficult to figure out how to get the program to update the light mode while it was also continuously running the current light mode. I eventually settled on running 2 programs. The first program creates a server using Flask that displays a simple static website with buttons that change the lights. If a button is pressed it updates a file with the new button mode, the second program is controlling the lights and checking if the file has been updated with a new mode to display.
I tried unsuccessfully to use multithreading in the program. However, I was never able to consistently get the lights to update.

Jeopardy Trivia Game

November 2019
Practice your trivia knowledge with trivia questions that have appeared on Jeopardy!!
GitHub: crhaugen/Jeopardy-Trivia


Jeopardy Trivia is an interactive web application using Node.js and Express that allows users to play a simplified version of jeopardy with questions that have appeared on the show.

Learned to use MongoDB (first time using a database) took awhile to figure out how to set up the database and add the question/answer pairs

First time creating a program that was asynchronous, so I spend many hours trying to debug and figure out why somethings weren't working correctly.
